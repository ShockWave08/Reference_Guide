2.How should we list a payout amount?

We should always use a "$" and also a comma seperator if the payout is above a thousand, (i.e $350.78, $3, 567.90, $21, 467. 30).

3. The type of loss in the system (green screen) and customer view is different?

Always use customer view to verify the type of loss and payout.

4. The insured is requesting a Letter of Experience be printed and mailed to them. How should I handle this request.

Save a copy of the a copy of the completed letter in the share drive BusOp_VendorSolutions\POSO Print\ LOE's and Coverage Letters.

5.The Service Request is requesting we create a loss history.

For these request we create a Household Level Letter.

6.If we have two losses on the same policy with the same date but different payout amounts, how do we list it?

We combine the payout amounts and list as one loss after verifying there have not been additional losses on the same date.

7.Should we use the mailing address, garaging address, or physical address when typing a letter for an auto?

Auto policies should always show the mailing address, if there is one.

8.We're asked to create a letter with information that has not been updated on the system yet.

Answer Will be here soon.

9.The insured's name shows differently between the client screen and declaration page, which should we use?

Always use either the name as it shows on Names in Facesheet (fire) or the name entered on the F/sheet Name if Diff on the Client/Driver Screen. If there is nothing in these fields , use the name as it has been entered on the policy.

10.The name on the policy screen has a trust; should i include the trust on my letter?

No. Only input the names of the insured as they would show on the declaration page. Don't add any additional verbiage, such as a trust, 'single man',or , 'sole and seperate property'.

11.What should we do if we're ask to create a letter showing the end date as a future date?

Go ahead and create the letter ending it on today's date. You may also want to include in the comment section that we do not create letter for future dates.

Driver Level Letters

Note: Please Contact Crystel Victor or Gerlyn Graneau for any information or updates you disagree with or you feel should be added.

1. Do we list all vehicles for the Primary and the Secondary insured, whether they're rated or not?

Yes! As long the insured is the Primary or Secondary insured on a policy, we can list the vehicle on the driver level letter.

2.We're asked to create a driver level letter with a specific vehicle that has aleady went out of force?

Create the Driver Level Letter using the vehicle cancellation date as the driver cancellation date.

3.What should we do if the driver is not rated on any vehicles in the IA screen.

In instances like these, we navigate to the Driver Assignment(DA) screen, then under the column named OVR, search in the driver row under this column for the letter R or S. R means the driver is a Primary rated owner and S meaning Secondary rated owner.

Vehicle Level Letters

Note: Please Contact Crystel Victor or Gerlyn Graneau for any information or updates you disagree with or you feel should be added.

1. Do we type LOE's for Trailers.

Yes, we will type LOE's for trailers. Use the vehice level letter ONLY for trailers. Also, use the serial number of the trailer as the VIN number. In your letter replace VIN with Serial manually, then add the serial number of the trailer.

3.What losses do we list when creating an LOE for a trailer?

List only losses associated with the trailer.
